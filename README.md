# cognixia-capstone

## General Info:

The application does the following: 
* GET /{any_number} returns the roman numeral equivalent of that number
* GET /ping returns the static text "pong".
* GET /version returns the version of the application.


## Technologies:

* AzureDevops
* Terraform
* Kubernetes
* Docker
